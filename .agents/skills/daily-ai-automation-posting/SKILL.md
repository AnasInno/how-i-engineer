# Daily AI Automation Posting

Use for social draft packaging around a completed automation.

## Workflow

1. Read the automation `README.md`, `IDEA.md`, and `VERIFY.md`.
2. Read `codex/daily-automation/POSTING-RULES.md`.
3. Draft X and LinkedIn posts under `drafts/daily-automation/<slug>/`.
4. Use repo/demo placeholders if URLs are not known.
5. For live posting, first run `python3 scripts/check_approval.py <slug> x-post` or `linkedin-post`.

## Voice

- concrete founder-led build note
- no hype words
- mention TeachClaw only as the serious flagship when relevant
- say what the tool does and what proof exists
