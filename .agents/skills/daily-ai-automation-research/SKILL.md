# Daily AI Automation Research

Use when choosing the next daily automation.

## Workflow

1. Read `codex/daily-automation/IDEA-FILTER.md`.
2. Generate specific workflow pains, not broad agent concepts.
3. Score each idea out of 10.
4. Keep only ideas with clear persona, input, output, and 2-minute demo path.
5. Reject anything platform-shaped or account-dependent.

## Output

For each candidate:

- title
- persona
- pain
- input
- output
- why it is one-day scope
- score
- first smoke test idea
