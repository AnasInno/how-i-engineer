# Idea Filter

Score every idea out of 10 before building.

## Must pass

- The title describes a specific task, not a category.
- The user can understand the value in one sentence.
- There is a concrete input and output.
- It can be built as a CLI or tiny local app.
- It can ship with fake/sample data.
- It does not require private credentials to demonstrate value.

## Scoring

Give 0-2 points for each:

1. Pain is boring and common.
2. Input/output are obvious.
3. Demo can run in under 2 minutes.
4. The tool saves real manual effort.
5. The post/story is clear without hype.

Build only if score is 7+.

## Good examples

- Turn meeting notes into a follow-up email and task list.
- Watch a folder of PDFs and make a daily summary digest.
- Convert messy WhatsApp order notes into invoice draft rows.
- Turn a CSV of job posts into tailored outreach snippets.
- Extract deadlines from screenshots/text and create reminder JSON.

## Reject examples

- AI sales agent.
- Personal productivity copilot.
- Autonomous research company.
- Generic inbox assistant.
- Anything that needs live Gmail/LinkedIn/X access for the demo.
